## Installation 
 - After download or clone run `npm install` to install all the dependancies.

## Frontend
  ## Switch To Frontend Folder
      cd editor
  ## Install required dependencies with:
      npm install
  ## Run with:
      npm start

## Backend
  ## Switch to Backend folder:
      cd editor-backend
  ## Install required dependencies with:
      npm install
  ## Run with:
      npm run dev
